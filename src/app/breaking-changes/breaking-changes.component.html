1. Description
  • Angular v16 requires node.js version v16 or v18 (Version Compatibility)
  • Angular v16 requires TypeScript version 4.9 or later (Version Compatibility)
  • In Angular 13, the view enginee has been removed.
    In 16, the Angular Compatibility Compiler (ngcc) has been removed
  • Updated behavior for Router.createUrlTree
    By creating the correct URLs, this sometimes resulted in different navigation behavior in the application. In the following case, the relativeTo property should be removed.
  • Angular Package Format changes
    -> Flattened ESM 2015 (FESM2015) outputs have been removed.
    -> EcmaScript 2020 outputs have been updated to EcmaScript 2022 (including the flattened output).
  • ReflectiveInjector has been removed. Use Injector.create to create an injector instead.
