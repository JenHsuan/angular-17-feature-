1. Class and InjectionToken router guards and resolvers was removed
Class and InjectionToken-based guards and resolvers are not as configurable, are less re-usable, require more boilerplate, cannot be defined inline with the route, and require more in-depth knowledge of Angular features (Injectable/providers). In short, they're less powerful and more cumbersome. (PR)
The following code will prompt
"Warning: implementing guard as injectable services is going to be deprecated in v15.2 and removed in v17"
We can update the code by the folowing way.
Or we can use functional route guard introduced in Angular v15.
2. The ripple properties of several Angular Material components were removed
The ripple property of MatButton, MatCheckbox, MatChip is deprecated. This change moves ripples to being a private implementation detail of the components.
