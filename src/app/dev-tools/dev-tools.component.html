Introduction
There are a few enhancements of the Angular dev tools.
  • View injected services of components
  • View the injector hierarchy and resolution paths
Installation
  • Extension for Chrome .
  • Addons for Firefox .
Inspectation
Open the Angular DevTools (Developer Tools -> Angular) and use the button on the left corner.
  • View injected services of components: select a particular component, see the dependencies under the "Injected Services" bar
  • View the injector hierarchy and resolution paths: select a specific injector
